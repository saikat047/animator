The project requires that your system has a few native applications installed already.

    - OpenKinect: http://openkinect.org/
    - OpenCV: http://opencv.org/

Ubuntu:
    Install 'libfreenect'.

Mac OS X:
    Install 'libusb-freenect'.
    Install 'libfreenect'.

Windows:
    Have no idea.

Build:
	The project uses Maven 3 to be built.
